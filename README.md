# projetoRoteirosViagensTelegram

Deploy no Render (bot polling):

- Tipo de servico: `Worker` (nao `Web Service`)
- Build Command: `pip install -r requirements.txt`
- Start Command: `python bot.py`

Variaveis minimas:

- `TELEGRAM_BOT_TOKEN`
- `PUBLIC_BACKEND_BASE`
- `MINIAPP_URL`
- `COST_ROTEIRO_COMMAND`
